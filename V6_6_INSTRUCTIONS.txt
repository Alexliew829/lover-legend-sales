Lover Legend Sales V6.6 自动更新版

这次只需覆盖 GitHub 网站文件。
Apps Script 继续使用目前已部署的 V6.5 Apps Script，不需要修改。

V6.6 功能：
1. 每次真正打开网页时，会从 Google Sheet 读取最新资料。
2. 手机 PWA 从后台回到前台时，会自动重新读取 Google Sheet。
3. 每次打开或回到 App，会自动检查 GitHub 是否有新版本。
4. 发现新版本后，Service Worker 自动切换并自动刷新一次。
5. 网络恢复时，会自动重新读取资料及继续同步未完成记录。
6. 离线时仍可使用缓存页面。

上传 GitHub：
- index.html
- style.css
- manifest.json
- sw.js
- version.json
- js/
- icons/
- images/

上传完成后：
1. 等 GitHub Pages 绿色勾。
2. 电脑第一次按 Ctrl + F5。
3. 手机完全关闭 App 后重新打开一次。
4. 如果手机仍卡在很旧版本，才删除旧 PWA 后重新安装一次。
之后的版本更新将自动检查，不应再经常需要手动刷新。
